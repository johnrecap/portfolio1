# ClinicFlow Manager

Interactive clinic operations dashboard demo built with React, TypeScript, Vite, Tailwind CSS, Zustand, Recharts, React Hook Form, Zod, and i18next.

The demo uses session-scoped mock data, so every visitor can add, edit, cancel, and reset records without affecting another visitor.

## Local commands

```bash
npm ci
npm run lint
npm run build
npm run dev
```

Production builds are emitted to `../../dist/demos/clinic-flow-manager` and served from `/demos/clinic-flow-manager/`.
