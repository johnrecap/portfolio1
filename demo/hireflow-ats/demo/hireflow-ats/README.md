# HireFlow ATS

HireFlow ATS is a professional, frontend-only Applicant Tracking System MVP. It demonstrates complex React patterns, beautiful aesthetic choices, and modern web application structure without relying on a backend.

## Features

- **Dashboard:** Interactive overview of recruitment metrics and candidate pipeline charts.
- **Candidate Pipeline (Kanban):** Fully functional drag-and-drop pipeline management using \`@dnd-kit/core\`.
- **Jobs & Candidates Management:** List views, search, and filtering of active requisitions and applicants.
- **Dark Mode & Theming:** Integrated dark mode using Tailwind CSS.
- **Localization (i18n):** English and Arabic (RTL) complete support using \`react-i18next\`.
- **Persistent State:** Uses Zustand with \`sessionStorage\` persistence for changes during testing.

## Tech Stack

- React 19 + Vite 6
- TypeScript
- Tailwind CSS v4
- Zustand
- Recharts
- DnD Kit
- React Router DOM
- i18next
- Lucide React
