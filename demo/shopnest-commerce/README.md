# ShopNest Commerce

ShopNest Commerce is a frontend-only commerce demo built by Mohamed Saied. It combines a public storefront with an admin workspace for products, orders, coupons, and store settings.

## Features

- Responsive storefront with product browsing, cart, checkout flow, and coupon handling.
- Admin dashboard for inventory, products, orders, and sales metrics.
- Mock data stored in the browser session so visitors can test changes independently.
- Arabic and English UI support with RTL-ready screens.
- Light and dark UI modes.

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, Zustand, React Router, React Hook Form, Zod, Recharts, Lucide React, and i18next.
